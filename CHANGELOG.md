## 0.0.1

- Initial release of easy_dio_client
- Added Dio-based HTTP client wrapper
- Supports GET, POST, PUT, PATCH, and DELETE requests
- Added token-based authentication using flutter_secure_storage
- Added file upload support (multipart/form-data)
- Added file download support
- Added response converter for mapping JSON to model classes
- Included example app with SplashScreen, Login, and Home flow
